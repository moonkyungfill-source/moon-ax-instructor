# 문경필 · AX 업무 자동화 강사 포트폴리오

단일 페이지 경력기술서 사이트.

🌐 **Live**: https://moon-ax-instructor.web.app

## 📄 구성

- `index.html` — 경력기술서 메인 (단일 페이지)
- `firebase.json` — Firebase Hosting 설정
- `.firebaserc` — Firebase 프로젝트 연결

## 🛠️ 기술 스택

- **Tailwind CSS** (CDN)
- **Pretendard · Lexend · Noto Sans KR** (Google Fonts)
- **Material Symbols Outlined** (아이콘)
- **Firebase Hosting** (배포)

## 🎨 디자인 시스템

feelsedu.com 디자인 시스템 기반 (오렌지/보라/시안 + 따뜻한 크림 배경).

| Token | Value |
|-------|-------|
| Primary | `#F97316` |
| Secondary | `#8B5CF6` |
| Accent | `#06B6D4` |
| Background | `#FFFBF5` |

## 🚀 로컬 개발

```bash
# 그냥 브라우저에서 index.html 열기
open index.html

# 또는 간단 서버
python -m http.server 8000
```

## 📦 배포

```bash
firebase deploy --only hosting:portfolio
```
